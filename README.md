# Premium Basic Calculator

A beautiful, modern web-based calculator built with vanilla HTML, CSS, and JavaScript.

## Features

- **Premium UI**: Glassmorphic design with a vibrant dark-mode gradient background.
- **Dynamic Resizing**: The calculator display text automatically resizes when the numbers get too long.
- **Keyboard Support**: Full keyboard support (NumPad, Enter, Esc, Backspace).
- **Responsive**: Clean layout built with CSS Grid and Flexbox.

## How to Run

Since this is a vanilla web application, you don't need to build any bundles or start any server.

1. Simply locate `index.html` in your file explorer.
2. Double-click to open it in your preferred web browser.

## Technologies Used

- **HTML5**: Page structure and semantic elements.
- **CSS3**: Variables, Flexbox, Grid, backdrop-filters, and transitions.
- **JavaScript**: ES6 Class-based architecture for clean state management and calculation logic.
